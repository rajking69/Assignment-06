# DigiTools – Digital Tools Buying Platform

DigiTools is a single-page web application where users can browse digital products, add items to a cart, remove items, and complete checkout with real-time feedback.

## Overview

This project demonstrates a basic e-commerce workflow using React. It focuses on component-based design, state management, and user interaction within a single-page interface.

## Technologies

* React 18
* Vite
* Tailwind CSS
* DaisyUI
* React Toastify
* React Icons
* Local JSON data

## Features

### Product and Cart Switching

Users can switch between product view and cart view using a toggle system.

### Cart Functionality

* Add items to cart
* Remove items from cart
* Prevent duplicate entries
* Update cart count dynamically
* Calculate total price

### Notifications

Toast messages provide feedback for user actions such as adding items, removing items, duplicate attempts, and checkout completion.
